# Sprint 0 — Governance Purification Status

## 1. Sprint Objective

Sprint 0 removes filtering-first governance and migrates the system toward classification-first institutional architecture.

The sprint objective is to purify upstream layers so they preserve opportunity distribution, expose classification richness, and leave allocation authority exclusively downstream.

## 2. Core Doctrine

- classification upstream
- allocation downstream
- Decision Engine = only allocation authority
